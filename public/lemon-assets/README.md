# Lemon Mascot Assets

This directory contains 3D rendered lemon character expressions generated by SDXL-turbo.

To generate these images, run:
```bash
npm run setup:lemon
```

The script will create 6 PNG files:
- lemon-neutral.png
- lemon-yes.png
- lemon-no.png
- lemon-probably.png  
- lemon-probably_not.png
- lemon-dont_know.png

Each image is a 3D rendered anthropomorphic lemonade glass character (Kool-Aid Man style) with different expressions.
